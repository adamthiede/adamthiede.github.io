Greetings, Internet Browser. Welcome to my homepage. I'm a vim editor, FOSS contributor, package manager, server racker, Pinephone caller, track[point|ball] mouser, ath9k wifi transmitter, Catholic believer, self-hoster, and secret keeper.

I run this website and a few others. I'm not a modern web developer, and I have strong opinions about what I put on my pages. That usually means no cookies, absolutely zero javascript (there's always a better way), and readable un-minified HTML and CSS. I get a kick out of writing HTML myself. This website respects your privacy!
