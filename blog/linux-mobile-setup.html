Linux Mobile Setup

[link—standalone]

This is a current snapshot of how I set up my mobile Linux system (Pine64 Pinephone, as of February 2022) for future reference and to help me collate all my information in one place. I usually forget something when I upgrade or reinstall the device.

I create my OS image with pmbootstrap. pmbootstrap offers a lot of choices. I choose edge, phosh, and set my locale to en_US.UTF-8, as God intended. I have a 2GB MicroSD card running JumpDrive to install PostmarketOS to the eMMC of the Pinephone with --sdcard=/dev/sda --fde.

After the system boots, I join it to the network, and install the following:
vim git htop rsync mpv ffmpeg urlscan newsboat phosh-antispam vvmplayer dino flatpak gnome-podcasts flatpak numberstation yad seahorse powersupply geary evolution foot dconf-editor yad
Next I add add flathub and my repo to flatpak remotes. I install my builds of Signal, Axolotl, and my Dice program (all detailed on my repo's site) and NhekoReborn, FluffyChat, KeePassXC, and Komikku. I install KeePassXC on from flatpak because it keeps QT libraries off the main system, and I'm installing them with NhekoReborn anyway.

I run the restore outlined in the "upgrade" detailed above from my encrypted "backup" sdcard, which I regularly rsync --delete my home directory to.

On PostmarketOS, I add the following cron entries:

# daytime
30 06 * * * /home/adam/bin/profile-swap restore
30 07 * * * /home/adam/bin/themeswitch.sh light
# nighttime
59 19 * * * /home/adam/bin/profile-swap store
59 19 * * * /home/adam/bin/profile-swap silent
30 19 * * * /home/adam/bin/themeswitch.sh dark
And speaking of cron, I use rc-update to add crond and waked to the default runlevel. Waked ensures the clock program will wake up the phone and sound the alarm when appropriate.

The keyboard case is fun. It's kind of broken and not really ideal - I'd like to to be thinner and larger - but it's awesome to have when on the go. At home I mostly don't use it because I can't plug the phone into my dock when the case is attached - power from 2 sources at once is bad. When using the keyboard I change the scaling from 200% to 150% or 175%. Phosh doesn't work well in landscape at all.

And that's about it. Future self, hope this helps.
