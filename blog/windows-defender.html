Windows Defender

[link—standalone] I'm always hearing about Windows 10 being awful - people don't like the forced updates, telemetry, ads in the start menu, etc, but in my experience it's not really that different than past windows versions once you get past the first boot.

Yeah, there's a lot to dislike about Windows 10 - the ads (especially if you pay for a key) are disrespectful, gross, and slimy, and they're everywhere - start menu, lock screen, notifications... endless ads! They really show you how much Microsoft values its desktop customers. Cortana is largely useless and I've never seen a single person actively engage with that feature. The built-in dark patterns leading you to enabling a Microsoft account for everything are pretty wild. The lack of bitlocker drive encryption on the home version is outrageous - you have to pay for the pro upgrade to use that.

Windows 10 isn't any more intrusive or workflow-breaking than 7. The telemetry is basically the same, but in 10 they actually expose some controls to the user, so many people think that it is the only Microsoft OS with any telemetry. (Side note - if you turn off telemetry in Windows 10, it will, after a time, turn it back on for you. Same with updates. You have no control.)
The update process is very similar, and I've seen Windows machines (7, 8, 10) reboot in the middle of work. The ads in the start menu is sort of new, but every Windows version tightens the screws a little bit, so it was just a matter of time. Each time you update, there are new little annoyances attached to your machine. And there's a new build every 6 months that breaks something new! These aren't isolated incedents, either - Windows 10 1809 was pulled from release twice, and even the Server 2019 builds were unavailable while Microsoft fixed this massive issue that their thousands of programmers and scores of "Windows Insider" testers couldn't catch.

I've opted out of using Windows anywhere but work, and using the enterprise version mitigates most of the bad stuff. I've seen comments from readers on Ars Technica (really!) and reddit where users defend Windows 7 as this bastion of privacy and security, like it's incapable of phoning home to Microsoft, and many people brag about disabling updates and using Windows 7 until long after EOL. Doesn't make any sense to me why, if you're not going to jump ship to macOS or Linux, why you'd continue to use a proprietary OS that you've made insecure.

Anyway. Windows rants always happen when Microsoft breaks RDP with a new update, which happens about twice a year.
by Adam Thiede
