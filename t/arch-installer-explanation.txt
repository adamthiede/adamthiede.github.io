The previous entry, <a href="https://elagost.com/w.php?t=arch-installer.txt">arch-installer.txt</a>, is an incomplete, possibly-wrong installation guide for Arch Linux. Follow at your own peril. It's not to be run as a script, just to have up in a tmux pane so I can remember what to do. The <a href="https://wiki.archlinux.org/index.php/Installation_guide">Arch Wiki's install guide</a> is great, but a little too much text for me when I just need a quick reference.

Generally, installing Arch Linux is not as hard as it's made out to be. It's roughly ten major steps, each with just a few commands.
<ol>
<li>Download the ISO, burn to a CD/USB, and boot from it</li>
<li>Partition your disks - /boot, /, swap (/home and /var?)</li>
<li>Optionally set up encryption at this point</li>
<li>Mount partitions and pacstrap (install base system + goodies)</li>
<li>Change Root (arch-chroot) and install any extras (pacman -S $PACKAGES)</li>
<li>Create user(s) and set up sudo</li>
<li>Enable services (systemctl enable $SERVICE)</li>
<li>Create ramdisk and install bootloader</li>
<li>Set locale and timezone</li>
<li>Exit, unmount partitions, and reboot into your new system!</li>
</ol>
Very easy (trivial in fact) for the seasoned Linux admin, not difficult for the long-time Linux user, and still quite concise enough for a newbie to follow.
Arch is kind of a meme distro, not super useful in the enterprise and much more work than necessary. But it scratches that special itch for me, and I really enjoy using it. That's not to say if Ubuntu had a rolling release I wouldn't use that. I'm all for using something that's easy to maintain, fairly popular, and genrally supported by most software and enterprise use cases. When Debian 10 comes out later this year I'll definitley be excited about that, but for now, I'll stick with the simplicity that Arch offers. I enjoy that it's un-opinionated (mostly just upstream), has every package under the sun, and is exactly what I want it to be - a simple rolling release with no weird customization and nothing added in but what I want. It feels like it's my own custom distro, and I like that.
