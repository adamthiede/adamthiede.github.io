Recently I have been working on a few software projects. Here is a summary of them.
  • AlephOne Marathon Launcher: remember Bungie's Marathon games? There's an open-source continuation located here but it requires the game to be run from the command line. Obviously I'm not terminal-averse, but this was fun to code. I also made a PPA for Ubuntu 18.04.
  • Link Saver: ever used Instapaper or Pocket? Basically a bookmark saver but you paste the URL in the terminal. It can save an offline file too. Pretty basic so far but I made it for me to use.
  • 2-factor Auth Anywhere: A portable set of ImageMagick scripts that basically took oathtool and put it into a little .jpg. I was working within the constraints of a) the device in question not having a real physical keyboard and b) the screen being low-res and small. It was more or less a fun exercise but nothing to take too seriously.
  • sfetch: a neofetch-like prompt that is MUCH smaller and lighter-weight. Made since I have a lot of systems I SSH into on a regular basis, and it's nice to have a little stats output right when I log in. As said in the repo, neofetch was too heavy for my poor-man's server fleet of raspberry pi zeros. This works better.

Mostly these are self-serving things that I made just to have fun!
