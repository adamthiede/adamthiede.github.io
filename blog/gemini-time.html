Gemini Time!

[link—standalone]

Attention: you should be using Gemini.

I think the indie web space is pretty cool, if you do it right, but the tempatation to make anything but a static website is very strong. Most people don't do it right. Web bloat - JavaScript frameworks that bring in piles of dependencies with security issues, style sheets that only work in Chrome, dynamic content that takes forever to load, tracking, ads, and all sorts of other wastes of bandwidth and electricity - these are all status quo on the modern web. Want to use an older computer? Sorry, your average news or recipe website will bring it to its knees. These sites should just serve text, and maybe an image or two, but they do not.

Why a new protocol?

Essentially, nobody but hyper-nerds avoids web bloat. Web bloat is making us throw away perfectly usable computers, create e-waste, killing our planet with useless energy consumption, giving power to massive surveillance corpos, and recklessly introducing security problems left and right. Web bloat is morally wrong. The Web worked fine without JS and CSS frameworks. It can work that way again. It should. But now that this is the status quo, companies won't give it up - turning what should be text-only sites into interactive web applications that track your usage and location and battery level and heartbeat and GPS location is too tempting to most webmasters, assisted of course by marketing departements. You almost never see web bloat on a regular blog or personal website.

With a new protocol that is intentionally limited, temptation is avoided. Terminal-based browsers are a first-class citizen. Text is downloaded near-instantly. There's no energy waste because 100% of the page's content is, well, actual content, not literal junk that nobody wants.

I've had a gemini capsule (that's what we call websites) set up on this server for a while: (gemini link). You will need a special browser to use it. I like Amfora, but if you must have a GUI, Lagrange is also fantastic.

The Gemini space is like what the web was like in the early days. Before powerful computers, before smartphones. It's beautiful and interesting and full of people - not bloat.

by Adam Thiede
