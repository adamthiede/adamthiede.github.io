Computer Software is a key component in liberating yourself from Big Tech.

The primary software component computers load is called the Operating System. It runs all the programs and controls the hardware. Operating Systems you may know are Windows, macOS, iOS, and Android. None of these, in their default state, remove Big Tech's thumb from your back.

If you are on your journey and not fully free yet, that's no reason to stop doing software updates. You should do all software updates as soon as you can, forever, unless you have the technical knowledge to explain why you won't do it. Apple has the exact same amount of control over a non-updated iPhone and an updated iPhone - the only difference is that the non-updated iPhone is far more vulnerable to attack. When updates are released, the security content of those updates is publicized, and attacks on out-of-date software become far easier.
If you have a proprietary device and don't update it, you're not sticking it to the man. You're only playing yourself.

General Guidelines on Software Usage

The general rule here is "do not use smartphone apps from the Apple App Store or the Google Play Store", but that's not possible all the time (how else will you get Signal on your iPhone?). Trusting these stores and the programs in them means trusting Big Tech to know what you install and track your usage patterns.

If the website works at all, even if it's slightly worse, don't use the app. Good web browsers help compartmentalize software from touching the sensitive data on your computer.

Stick to F-Droid only, if you use Android.

Operating Systems

Laptops and Desktops

For regular laptops and desktops, you can't get pick a more accessible choice than Pop!_OS by System76. They sell computers pre-loaded with Pop!_OS. None of your Windows or macOS programs will run on Pop!_OS. It is a Linux-based operating system. Linux runs on most of the world's web servers and all of the world's supercomputers. It is the primary choice for secure and high-performance workloads for technical users. Thankfully, Pop!_OS is super accessible for regular users, and will serve the needs of most much better than Windows or macOS.

You can, and should, replace your current operating system.

  • Download Pop!_OS - it comes in an ISO image format
  • Grab a USB stick that's 4GB or larger
  • Download etcher, software for writing Operating System images to USB disks
  • Write Pop!_OS's ISO file to the USB stick.
  • Power off the target computer and plug in the USB stick
  • Turn on your computer, and quickly press the key to bring up the boot menu. This varies based on manufacturer but you can always power back off and try again.
  • Common boot menu keys are the function keys F9 (HP), F12 (Dell/Lenovo), and F10.

Smartphones

Smartphones are entirely different. Most operating systems for smartphones have to be built for a specific model and are less secure than the regular one. iPhones are a lost cause - Android's the only way to not have your smartphone usage tracked and beholden to Google or Apple, ironically. That's because Android's core is Open Source and built on Linux as well.

If you have a supported smartphone, you can use CalyxOS - it comes completely Google-free, has a built in firewall, and an alternative app store that doesn't require logging in and has only trustworthy, private software. GrapheneOS is much more secure, but also much less user friendly, and harder to install. Both work best on Google Pixel 4a or 3a, which are rather inexpensive. LineageOS is another option that works on a much wider range of devices, but does not have the security, privacy, or usability benefits that the previous two do.

For more advanced software needs, unfortunately this is where Big Tech still has you under their thumb. Adobe and Autodesk products only work on Big Tech Operating Systems. There are always Free and Open Source alternatives to these specific proprietary programs, but they may not be quite as high quality, or the workflow might be different. It all comes down to compromise. You must choose - do you want say "Goodbye, Big Tech", or do you not?

Advanced

Other Operating System recommendations below for the more technical - all of these are completely big-tech-free.

  • Debian has been around for the longest of any Operating System and is incredibly stable and lightweight.
  • Qubes is for the ultra-paranoid, and runs everything in isolated subsystems.
  • VirtualBox is a program that can run a Windows as a program in Linux (computer within a computer), with large performance tradeoffs. If you have a specific program you must use Windows for, but it's not very resource intensive, run Windows in VirtualBox, and run the program in there.
  • Wine can run Windows software natively on Linux and macOS.
  • Bottles is an easy way to manage Wine and programs run with Wine. Easier and less overhead than running a full Virtual Windows OS.
